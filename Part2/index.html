Next steps

I continue to follow the 'full stack developer' course provided by freeCodeCamp. Whilst I have found this course fun and interactive, there is a growing part of me which is missing the excitement of actually programming something.

freeCodeCamp describred the learning order in the course like constructing a house, with HTML being the structure, CSS being the decoration and Java Script (or another development system) being the interactivty (e.g. the power supply enabling you to turn lights on and off).

This order makes sense to me and I will continue to trust in the program, although I may go rogue outside the hours I've scheduled for structured learning and try to complete a few things in Java Script.

But the next step for me in relation to my core development through freeCodeCamp is CSS!
